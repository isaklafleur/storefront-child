Plugin integrates your Wordpress site with your instance of mlm-soft.com cloud platform.

== Description ==
Plugin allows:
- single authorization and registration of users both website and mlm-soft.com platform
- referral links and personalized websites for distributors
- additional integration with Paid Memberships Pro and Woocommerce

== Installation ==
In your WordPress admin, go to Plugins > Add New to install MLM Soft Integration, or:

1. Download the latest version of the plugin.
2. Unzip the downloaded file to your computer.
3. Upload the /mlmsoft/ directory to the /wp-content/plugins/ directory of your site.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Set up options on the MLM Soft options page in your WordPress admin. Required options are: MLM Soft Project URL, Online Office URL, API Token. If you have any questions please ask help@mlm-soft.com
